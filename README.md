## Project Overview
COMP3015-CW2 combines advanced shader techniques with a simple gameplay loop.

The player explores a terrain environment, enters a hut, collects an Energy Cell, and uses it to activate a reactor. The scene uses shadow mapping, HDR, fog, textured models, procedural rocks, collisions, and in-game UI text.
